# The Movie DB Apps

This app is an implementation of The Movie DB API

## Tech stack:
1. Kotlin 2.3.21
2. MVVM
3. Jetpack Compose with Nav3 - UI
4. Retrofit - Rest framework
5. Hilt - Dependency Injection
6. Room - persistent storage
7. Json DataStore - persistent configuration storage

## How to run
1. Make sure the AGP and JDK is set
2. add `secret.properties` alongside `local.properties`
3. add `TMDB_API_KEY` of your own
4. build the app
